Проект: Telegram бот для анализа авто

Цель
Бот принимает ссылку на авто → определяет источник → парсит данные → выдаёт структурированный отчёт в Telegram.
Источники данных

Encar (через API, car ID из URL)
KB차차차
Kkar

Данные для сбора

Модель, год, пробег, цена, цвет, регион
성능점검기록부 (состояние: ДТП, замены, ржавчина) — закодирован в JSON, нужна расшифровка
보험이력 (история страховых случаев)

Стек

Telegraf.js + TypeScript
Node.js
Хостинг: Fly.io

Архитектура
/src
  /scrapers
    encar.ts
    kbcha.ts
    kkar.ts
  /formatters
    report.ts
  router.ts
  bot.ts
Порядок разработки

Ядро Encar (API, два endpoint'а, расшифровка 성능점검기록부)
Ядра KB차차차 и Kkar
Router (парсинг URL → выбор ядра)
Formatter (единый формат отчёта)
Telegraf бот
Деплой на Fly.io

Следующий шаг
Ты показываешь JSON ответы от Encar API → строим типы и парсер.
